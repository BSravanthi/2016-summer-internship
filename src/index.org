** Policies
   + Reporting Hours :: 9am-6pm (9 hrs including 1 hr lunch break)
   + Leaves :: No leaves are granted unless it is an emergency.  The approval
        for an emergency leave is taken from Prof. Venkatesh Choppella.
   + Work status reporting
     - Each intern creates an own repository named =vlabs-internship=
     - Each intern provides a daily log of work done in this [[./daily-work-log-template.org][format]].  The
       report(file: daily-log.org) is pushed to the individual's
       =vlabs-internship= repository at the end of the day.
   + Each intern make two presentations tentatively on [2016-06-13 Mon] and
     [2016-07-14 Thu].  on work accomplished till that point in time.
   + Each intern submits a final report as a document on his/her
     accomplishments.
   + Containers or VMs created on the Virtual Labs infrastructure are strictly
     used only for the purpose of realization of the tasks assigned to the
     intern.
